# estomania-server
server for estomania

# Installing
- Install all the modules by running following command:

`npm run install`

# Start the server
- Start the server by running the following command:

`npm run start`

# Private publish

npm publish --access-restricted

# Start the server in hot-reload (dev) mode:

`npm run dev`
